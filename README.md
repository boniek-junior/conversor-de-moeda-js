# 💱 Conversor de Moedas
---

## 📌 Sobre o projeto

Projeto desenvolvido em **JavaScript** com o objetivo de converter moedas internacionais  
(**Euro, Dólar e Libra Esterlina**) para **Real (R$)**.

Criado durante a trilha de aprendizado da **Rocketseat**, com foco em:

- Lógica de programação  
- Manipulação do DOM  
- Interatividade com JavaScript  

---

## 🚀 Funcionalidades

✔ Conversão de moedas em tempo real (simulada)  
✔ Interface simples e intuitiva  
✔ Atualização dinâmica do valor convertido  
✔ Interação com eventos em JavaScript puro  

### 💵 Moedas disponíveis

- 🇺🇸 **Dólar (USD)** → BRL  
- 🇪🇺 **Euro (EUR)** → BRL  
- 🇬🇧 **Libra Esterlina (GBP)** → BRL

## ⚙️ Como funciona

1. O usuário insere um valor em moeda estrangeira
2. Seleciona a moeda desejada
3. O sistema realiza a conversão para **Real (R$)**
4. O resultado é exibido automaticamente na tela
